## 2. Description of Service

Liofa - a Discord moderation bot focused on moderating non-English languages and helping servers manage multilingual communities.

- **Language Detection:** Automatically detects message languages using the `cld` library
- **Content Filtering:** Filters messages based on customizable word lists, emojis, and language settings
- **Server Configuration:** Customizable settings for each server, including prefixes, whitelists, filtered terms, and channel-specific rules
- **Permission Management:** Role-based access control for bot commands and features
- **Moderation Tools:** Warning systems, watchlists, and automated moderation actions with configurable expiration periods
- **Logging:** Tracks moderation actions and filtered content for server administrators
